# Task 1: Basic Mathematical Operations

## Description
This program takes two integers as input from the user and performs basic arithmetic operations.

## Operations Performed
- Addition
- Subtraction
- Multiplication
- Division

## Special Condition
If the second number is zero, the program prevents division and displays:
"Division by zero is not allowed"

## Concepts Used
- Type casting using `int()`
- Arithmetic operators (`+`, `-`, `*`, `/`)
- Conditional statements (`if-else`)
- User input handling

# Task 2: Personalized Greeting

## Description
This program takes the user's first name and last name as input and displays a personalized welcome message.

## What This Program Does
- Accepts first name from the user
- Accepts last name from the user
- Combines both names
- Prints a formatted welcome message

## Concepts Used
- `input()` function
- String handling
- `print()` function
- `sep` parameter in print
